# GymBro 💪

**GymBro** est une application mobile de mise en relation entre sportifs fréquentant les salles de sport à La Réunion.

L'idée est simple : trouver un partenaire d'entraînement qui correspond à ton profil, ton niveau et tes objectifs.

## Concept

- Découvre des profils de sportifs proches de toi
- Matche avec ceux qui partagent tes objectifs
- Discute directement dans l'app pour organiser vos séances

## Stack technique

- **React Native** avec **Expo**
- **TypeScript**
- Navigation par fichiers (Expo Router)

## Lancer le projet

```bash
npm install
npm run start
```

> ⚠️ MVP avec données fictives uniquement (pas de backend actif).
